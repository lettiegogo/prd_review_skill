# 通用 PRD 审核标准 (基础版)

> 本文件为流程跑通用的占位标准。后续可替换为垂直领域详细标准。

## 1. 结构完整性
- [ ] 是否有明确的 **背景与目标**？
- [ ] 是否列出了 **需求范围**（In-Scope / Out-of-Scope）？
- [ ] 是否有 **角色权限** 定义？
- [ ] 是否有 **修订记录**？

## 2. 逻辑闭环
- [ ] 主流程是否清晰（用户视角操作步骤）？
- [ ] 是否有 **异常流程** 说明（如：网络断开、权限不足、数据为空）？
- [ ] 状态机是否闭环（如：草稿->审核->生效->失效）？

## 3. 字段与交互定义
- [ ] 表单字段是否有 **类型、长度、校验规则**？
- [ ] 列表页是否有 **排序、筛选、分页** 说明？
- [ ] 按钮/操作是否有 **前置条件** 和 **后置反馈**？

## 4. 数据与埋点
- [ ] 是否定义了需要统计的 **核心指标**？
- [ ] 是否有 **埋点需求** 说明？
